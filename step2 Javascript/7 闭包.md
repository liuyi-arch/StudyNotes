## 一、词法作用域与动态作用域

### JavaScript 使用词法作用域（静态作用域）

> 其他语言比如bash就使用的是动态作用域；

- 词法作用域决定于 **函数定义时** 所处的环境，而不是调用位置

```js
var value = 1;

function foo() {
    console.log(value);
}

function bar() {
    var value = 2;
    foo();
}

bar();
```

**结果是：** `1`

------

### 动态作用域

在动态作用域中：函数内部的变量查找，是从 **调用它的位置** 开始向上查找。

假如 JavaScript 是动态作用域，上面的例子结果会是`2`
